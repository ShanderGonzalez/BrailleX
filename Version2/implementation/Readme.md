# Implementacion
En esta parte se establece el orden de estructura de las carpetas que se llevo a cabo para el desarrollo del producto software "BrailleX"

## Estructura de Carpetas

```
implementation
├── app
│   ├── services
│   │   ├── traductor.js
│   │   └── validate.js
│   ├── utilities
│   │   ├── dictionary.js
│   │   └── limpiar.js
│   └── views
│       ├── brailleEs.html
│       ├── Equipo.html
│       ├── index.html
│       └── MisionVision.html
├── docs
├── images
├── node_modules
└── templates
    ├── css
    │   ├── estilosConocenos.css
    │   └── estilosPrincipal.css
    └── fonts
```
